XDC Datachain Gateway for Woocommerce via XDCPay browser plugin.

== Description ==
If you want to receive Woocommerce payments in the Datachain Foundation (DC) XRC20 token via [XDCPay](https://chrome.google.com/webstore/detail/xdcpay/bocpokimicclpaiekenaeelehdjllofo) Chrome Plugin this plugin is exactly what you need.
With simple installation and setup you are able to receive DC tokens on seconds.
DC token smart contract address: [xdc20b59e6c5deb7d7ced2ca823c6ca81dd3f7e9a3a](https://explorer.xinfin.network/tokens/xdc20b59e6c5deb7d7ced2ca823c6ca81dd3f7e9a3a#token-transfer).

== Installation ==

1. Goto your wordpress backend
2. Navigate to Plugins > Add new
3. Search for "XDC Datachain Payment Gateway"
4. "Install"
5. In the Woocommerce > Settings > Payments > Datachain Payment Gateway set field "Receive address" where you want to receive payments.
6. You are ready to receive DC token payments via XDCPay!
